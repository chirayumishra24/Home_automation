Goals & Prerequisites

Learning Objectives

By the end of this project, you will be able to identify electronic components, read and draw circuits, design automation systems for home or school, select and program sensors, and install/test systems.

Video & Interactive

Watch a short tutorial on electrical symbols: YouTube — electrical symbols

Learning Outcomes
  1. Identify electronic components used in automation systems.
  2. Read and draw electronic circuits.
  3. Design an automation system for a home or a school.
  4. Select appropriate sensors and program them.
  5. Install automation systems and test them.
Prerequisites

Understand simple circuits, how to draw/read circuit diagrams, and basic component functions: battery, resistor, switch, diode, LED, capacitor, earthing.
